## Day 2 — Adding & Subtracting Complex Numbers

---

### *"Combine like with like — it's simpler than you think"*

---

### Learning Objectives

By the end of Day 2, you will be able to:

- Add two or more complex numbers together
- Subtract one complex number from another
- Correctly identify and group **real** and **imaginary** parts
- Simplify results into **standard form** (a + bi)
- Avoid the most common mistakes students make

---

### Part 1 — The Golden Rule of Complex Arithmetic

Before touching a single sum, burn this rule into your memory:

> **Real parts combine ONLY with real parts.**
> **Imaginary parts combine ONLY with imaginary parts.**
> **They NEVER mix.**

This is exactly like combining **like terms** in algebra. You would never add 3x and 5y together — they are different types. In the same way, real parts and imaginary parts are different types.

$$3x + 5y \quad \text{stays as} \quad 3x + 5y \qquad \text{(you cannot simplify further)}$$

$$3 + 5i \quad \text{stays as} \quad 3 + 5i \qquad \text{(real and imaginary stay separate)}$$

---

### Part 2 — Adding Complex Numbers

### The Formula

$$\boxed{(a + bi) + (c + di) = (a + c) + (b + d)i}$$

**Step-by-step method:**
1. Write both complex numbers next to each other
2. Group the real parts together
3. Group the imaginary parts together
4. Add each group separately
5. Write your answer in standard form a + bi

---

### Worked Example 1 — Simple Addition

$$\text{Calculate } (3 + 2i) + (1 + 5i)$$

**Step 1:** Write them side by side:
$$(3 + 2i) + (1 + 5i)$$

**Step 2:** Group real parts and imaginary parts:
$$= (3 + 1) + (2 + 5)i$$

**Step 3:** Add each group:
$$= 4 + 7i$$

**Answer: 4 + 7i** ✓

---

### Worked Example 2 — Addition with Negative Imaginary Part

$$\text{Calculate } (5 + 3i) + (2 - 7i)$$

**Step 1:** Group the parts:
$$= (5 + 2) + (3 + (-7))i$$

**Step 2:** Simplify:
$$= 7 + (-4)i$$

**Step 3:** Write in standard form:
$$= 7 - 4i$$

**Answer: 7 − 4i** ✓

> 💡 **Note:** 7 + (−4)i and 7 − 4i are the same thing. Always write the cleaner version.

---

### Worked Example 3 — Adding Three Complex Numbers

$$\text{Calculate } (2 + i) + (3 - 4i) + (-1 + 2i)$$

**Step 1:** Group all real parts and all imaginary parts:
$$= (2 + 3 + (-1)) + (1 + (-4) + 2)i$$

**Step 2:** Simplify each group:
$$= (4) + (-1)i$$

**Step 3:** Standard form:
$$= 4 - i$$

**Answer: 4 − i** ✓

> 💡 **Note:** −1i is always written as simply **−i**. The coefficient of 1 is invisible.

---

### Part 3 — Subtracting Complex Numbers

### The Formula

$$\boxed{(a + bi) - (c + di) = (a - c) + (b - d)i}$$

**Step-by-step method:**
1. Distribute the minus sign to BOTH parts of the second complex number
2. Group real parts together
3. Group imaginary parts together
4. Subtract each group
5. Write answer in standard form

---

### Worked Example 4 — Simple Subtraction

$$\text{Calculate } (6 + 5i) - (2 + 3i)$$

**Step 1:** Distribute the minus sign:
$$= 6 + 5i - 2 - 3i$$

**Step 2:** Group the parts:
$$= (6 - 2) + (5 - 3)i$$

**Step 3:** Simplify:
$$= 4 + 2i$$

**Answer: 4 + 2i** ✓

---

### Worked Example 5 — Subtraction Producing Negative Parts

$$\text{Calculate } (3 + 2i) - (7 + 6i)$$

**Step 1:** Distribute the minus sign:
$$= 3 + 2i - 7 - 6i$$

**Step 2:** Group the parts:
$$= (3 - 7) + (2 - 6)i$$

**Step 3:** Simplify:
$$= -4 + (-4)i$$

**Step 4:** Standard form:
$$= -4 - 4i$$

**Answer: −4 − 4i** ✓

---

### Worked Example 6 — The Tricky Minus Sign ⚠️

$$\text{Calculate } (4 + 3i) - (1 - 5i)$$

**Step 1:** Distribute the minus sign carefully — it hits BOTH terms:
$$= 4 + 3i - 1 + 5i$$

> ⚠️ **Watch out!** Subtracting (1 − 5i) means:
> - −1 × 1 = **−1** ✓
> - −1 × (−5i) = **+5i** ✓ (minus × minus = plus!)

**Step 2:** Group the parts:
$$= (4 - 1) + (3 + 5)i$$

**Step 3:** Simplify:
$$= 3 + 8i$$

**Answer: 3 + 8i** ✓

---

### Part 4 — The Analogy: Different Currencies

Think of real parts and imaginary parts as **two different currencies** — say, **Pounds (£)** and **Dollars ($)**.

- The real part is your **Pounds** balance
- The imaginary part is your **Dollars** balance

| Operation | Pounds (real) | Dollars (imaginary) | Result |
|-----------|--------------|---------------------|--------|
| Start with | £3 | $2i | 3 + 2i |
| Add | £1 | $5i | 1 + 5i |
| **Total** | **£4** | **$7i** | **4 + 7i** |

You would never add £3 and $5 together into a single amount — they're different currencies. Exactly the same logic applies here.

> **Subtracting** is like spending money: if you spend £7 when you only have £3, you go into **negative** Pounds (−4). The same can happen with imaginary parts — the result can be negative.

---

### Part 5 — Special Cases Worth Knowing

### Case 1 — Adding a real number to a complex number

$$5 + (3 + 2i) = (5 + 3) + 2i = 8 + 2i$$

Think of 5 as the complex number (5 + 0i) — it only affects the real part.

---

### Case 2 — Adding a purely imaginary number to a complex number

$$(4 + 3i) + 7i = 4 + (3 + 7)i = 4 + 10i$$

Think of 7i as the complex number (0 + 7i) — it only affects the imaginary part.

---

### Case 3 — A complex number minus itself

$$(a + bi) - (a + bi) = (a - a) + (b - b)i = 0 + 0i = 0$$

Any complex number subtracted from itself gives **zero** — just like real numbers.

---

### Case 4 — Results that are purely real or purely imaginary

$$(3 + 4i) + (1 - 4i) = (3+1) + (4-4)i = 4 + 0i = 4$$

When imaginary parts cancel, the result is a **real number**. This is perfectly valid — remember, real numbers are a subset of complex numbers!

$$(3 + 4i) - (3 - 2i) = (3-3) + (4-(-2))i = 0 + 6i = 6i$$

When real parts cancel, the result is **purely imaginary**.

---

### Part 6 — Geometric Interpretation (Sneak Preview of Day 5)

On an Argand diagram, complex numbers are **points** (or **vectors**) in a 2D plane. Adding complex numbers is exactly like **adding vectors**:

- **(3 + 2i)** means: go 3 right, 2 up
- **(1 + 5i)** means: go 1 right, 5 up
- **Sum (4 + 7i)** means: go 4 right, 7 up — the combined journey!

This is identical to **vector addition** in physics and maths. You'll explore this visually on Day 5.

---

### Part 7 — Key Rules to Memorise

$$\boxed{(a + bi) + (c + di) = (a+c) + (b+d)i}$$

$$\boxed{(a + bi) - (c + di) = (a-c) + (b-d)i}$$

$$\text{Real parts} + \text{Real parts} \quad \text{ONLY}$$

$$\text{Imaginary parts} + \text{Imaginary parts} \quad \text{ONLY}$$

$$\text{Never mix real and imaginary parts together}$$

---

### Part 8 — Common Mistakes to Avoid

| Mistake | Wrong | Correct |
|---------|-------|---------|
| Mixing real and imaginary | (3 + 4i) + (1 + 2i) = 10i | = **4 + 6i** |
| Forgetting to distribute minus | (5+3i) − (2−4i) = 3 − i | = **3 + 7i** |
| Dropping imaginary part | (2 + 0i) + (1 + 3i) = 3 | = **3 + 3i** |
| Writing bi as the imaginary part | Im(3+4i) = 4i | = **4** (just the number) |
| Leaving −1i as −1i | answer = −1i | write as **−i** |

### Practice Problems

Try these yourself before checking answers!

**Level 1 — Basic**

1. (2 + 3i) + (4 + 5i)
2. (7 + i) + (3 + 2i)
3. (6 + 4i) − (2 + 1i)

**Level 2 — Intermediate**

5. (−3 + 7i) + (5 − 7i)
6. (4 + 2i) − (−1 + 3i)
7. (2 + 3i) + (−2 − 3i)
8. (1 + i) + (2 + 2i) + (3 + 3i)

**Level 3 — Challenge**

9. Find z if z + (3 − 2i) = 7 + 5i
10. Find w if (4 + 6i) − w = 1 + 2i
11. Simplify: (a + bi) + (a − bi)
12. If z₁ = 3 + 4i and z₂ = 1 − 2i, find z₁ + z₂ and z₁ − z₂, then verify that (z₁ + z₂) + (z₁ − z₂) = 2z₁

---

### Answers

**Level 1**
1. (2+4) + (3+5)i = **6 + 8i**
2. (7+3) + (1+2)i = **10 + 3i**
3. (6−2) + (4−1)i = **4 + 3i**
4. (5−3) + (−2−(−6))i = 2 + 4i = **2 + 4i**

**Level 2**

5. (−3+5) + (7+(−7))i = 2 + 0i = **2**
6. (4−(−1)) + (2−3)i = 5 + (−1)i = **5 − i**
7. (2+(−2)) + (3+(−3))i = 0 + 0i = **0**
8. (1+2+3) + (1+2+3)i = **6 + 6i**

**Level 3**

9. z = (7 + 5i) − (3 − 2i) = (7−3) + (5−(−2))i = **4 + 7i**
10. w = (4 + 6i) − (1 + 2i) = (4−1) + (6−2)i = **3 + 4i**
11. (a + bi) + (a − bi) = (a+a) + (b+(−b))i = **2a** — a real number! (This is why conjugates are so powerful — Day 4!)
12. z₁ + z₂ = (3+1) + (4+(−2))i = **4 + 2i**; z₁ − z₂ = (3−1) + (4−(−2))i = **2 + 6i**; Verify: (4+2i) + (2+6i) = 6 + 8i = 2(3+4i) = 2z₁ ✓

### Day 2 Summary

| Concept | Key Takeaway |
|---------|--------------|
| Golden rule | Real with real, imaginary with imaginary — never mix |
| Addition formula | (a+bi) + (c+di) = (a+c) + (b+d)i |
| Subtraction formula | (a+bi) − (c+di) = (a−c) + (b−d)i |
| Minus sign warning | Distributes to BOTH parts of the second number |
| Geometric meaning | Same as adding 2D vectors on the Argand diagram |
| Special result | If imaginary parts cancel → purely real answer |
| Special result | If real parts cancel → purely imaginary answer |
